## Installation


### SETUP
1) cd app
2) sudo gem install sass
3) npm install
4) pipenv install
5) pipenv shell
- if error on mac run `brew reinstall pipenv` then re run `pipenv shell`

### RUN
1) pipenv shell
2) python app.py

Serving on http://127.0.0.1:5500

### Remove pipenv
pipenv --rm


### install and save pipenv libraries
pipenv install {library}


### Exit APP
ctl + C

### Exit Pipenv
ctl + D
